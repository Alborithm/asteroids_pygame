# Asteroids game done for Boot.dev course

This is an asteroids game done for a course in boot.dev

## Tech used
- Python3 (version 3.13)
- pygame (version 2.6.1)

## Requisites to run game
- Install python (see below)
- instal uv (virtual environment): https://docs.astral.sh/uv/getting-started/installation/
- install pygame: https://www.pygame.org/wiki/GettingStarted

### python installation guide
for Windows (WSL 2) / Linux Users
If you're on Linux (or Windows WSL 2), you may already have Python installed. You can verify this by running:

`python3 --version`

If that worked, you can skip ahead!

If it didn't, we'll use the default package manager, apt, to download python.

Run this command to install python3:

`sudo apt update`
`sudo apt install -y python3`


Make sure to read the installation logs in case there's some step you're missing!

Restart your shell to make sure the changes have taken effect.
Verify the installation:

`python3 --version`

You should get back a valid Python 3.x version.
